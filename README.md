# N7x - Roblox GUI Script

سكريبت متقدم لـ Roblox يوفر واجهة مستخدم احترافية مع أدوات تعديل اللعبة.

## ✨ الميزات

### الأوامر الرئيسية
- ✈️ **نظام الطيران** - طيران بكل الاتجاهات مع التحكم الكامل بالسرعة
- 🚫 **اختراق الجدران (Noclip)** - اخترق أي حاجز في اللعبة
- 📍 **نقاط الحفظ** - احفظ موقعك والعودة إليه في أي وقت
- 🎯 **الانتقال للاعبين** - انتقل إلى أي لاعب في اللعبة

### أدوات اللاعب
- 👁️ **كشف اللاعبين (ESP)** - رؤية جميع اللاعبين من خلال الجدران
- 📹 **مشاهدة شاشة اللاعب** - رؤية ما يراه لاعب آخر من وجهة نظره
- 🔍 **البحث عن اللاعبين** - ابحث عن أي لاعب برقم أو اسم
- 📸 **صور اللاعبين** - عرض صورة اللاعب المختار

### التحكم بالطيران
- **W** - للأمام
- **S** - للخلف
- **A** - لليسار
- **D** - لليمين
- **Space** - للصعود
